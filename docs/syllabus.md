# Data Mining Course Syllabus (Reference)

## Institution
Tunis Business School, University of Tunis

## Course
- Title: Data Mining
- Academic year: 2025-2026

## Course Overview
This course introduces core machine learning and data mining concepts, including:
- classification and prediction,
- data visualization,
- evaluation and model performance,
- neural networks and boosting.

Students are expected to build practical intuition and a formal understanding of when and why methods work.

## Prerequisites
Basic algebra, statistics, and Python or R programming.

## Learning Outcomes
By the end of the course, students should be able to:
1. Explore data and identify characteristics that affect model choice.
2. Understand and apply a wide range of learning algorithms.
3. Implement training, validation, and testing workflows.
4. Perform feature engineering and feature selection.
5. Recognize and address overfitting/underfitting.
6. Evaluate model performance using suitable metrics.
7. Select methods appropriate for real-world problems.
8. Distinguish supervised from unsupervised learning.
9. Explain how boosting can improve model performance.
10. Use Python/R ML libraries effectively.
11. Adapt to new tools and technologies.

## Course Topics (Tentative)
- KNN for regression/classification
- Naive Bayes
- Decision Trees
- Random Forest
- Logistic Regression
- Neural Networks
- Model Performance and Evaluation
